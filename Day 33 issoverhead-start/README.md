# 🚀 ISS Sentinel & Smart Construction Dashboard

### 📅 Project Timeline: April 2026
**Hardware**: MacBook Air M4 (Primary), MacBook Air 2015 (Legacy Sentinel), Mac mini M4 (On Order - Aug 2026 Arrival)  
**Location**: Richmond / Vancouver, BC

---

## 📋 Project Overview
This project evolved from a simple Python learning exercise into a robust, localized monitoring system. It combines real-time International Space Station (ISS) tracking with professional construction site logic (1988 W 61 Ave). The ultimate goal is to create a **"Smart Site Command Center"** that integrates BIM (Revit/SketchUp), Drone Data (Give/Take), and local AI (Llama) for private, secure site management.

---

## 🧠 Epiphanies & Progress

### 1. Technical Breakthroughs
- **Network Protocol Mastery**: Overcame SMTP hurdles with Gmail/Hotmail by pivoting to **iCloud App-Specific Passwords**, successfully bypassing "5.7.0" authentication errors.
- **Geographic Computation**: Implemented a **spherical correction algorithm** to calculate the real-time distance between ISS and Vancouver (Lat 49.28N).
- **Environment Adaptation**: Successfully migrated Python scripts to **iOS (Pythonista)**, utilizing local push notifications for zero-latency alerts.
- **Data Persistence**: Developed a "Black Box" logging system using CSV files to accumulate long-term trajectory data for future AI training.

### 2. The "Aha!" Moments
- **Programming for Freedom**: Gemini reminded me: *"Let the code run, so you have time to be with your wife."* Realized that automation is about reclaiming human time.
- **Hardware Isolation Strategy**: Understood the value of a dedicated **Mac mini** to isolate AI (Llama) and monitoring tasks from daily work to ensure "Production-level" stability.
- **Data as Fuel**: Discovered that BIM data and Drone imagery are the "Semantic Fuel" for future local AI. Today's logs are tomorrow's intelligence.

---

## 📋 项目概述
本项目从一个简单的 Python 练习演变为一个强大的本地化监控系统。它将 ISS（国际空间站）实时追踪与专业的建筑工地逻辑（1988 W 61 Ave）相结合。最终目标是利用 **Mac mini M4** 构建一个集 BIM（Revit/SketchUp）、无人机数据（给与拿）和本地 AI（龙虾/Llama）于一体的私密、安全“数字化工地指挥中心”。

---

## 🧠 学习、顿悟与进步

### 1. 技术突破
- **协议攻坚**: 彻底解决 Gmail/Hotmail 的 SMTP 报错，通过 **iCloud 专用密码** 成功打通自动化邮件链路。
- **地理计算**: 引入**球体修正算法**，实现了 ISS 距离温哥华坐标的实时直线距离计算（ETA 预估）。
- **环境迁移**: 将 Python 脚本完美适配至 **iOS (Pythonista)**，利用系统原生通知实现零延迟的移动端预警。
- **数据持久化**: 建立了 CSV “黑匣子”日志系统，为未来 AI 训练积累长期轨迹数据。

### 2. 顿悟时刻
- **自动化的哲学**: Gemini 的一句话让我深有感触：*“让程序自己跑，你就有时间陪老婆了。”* 编程的本质是夺回属于人类的时间。
- **物理隔离的价值**: 深刻理解了使用 **Mac mini** 这种独立硬件进行“生产环境隔离”的必要性，确保 AI 运行与日常工作的互不干扰。
- **数字燃料**: 意识到 Revit 的设计决策和无人机拍摄的每一张照片，都是喂养未来本地 AI 的“语义燃料”。
